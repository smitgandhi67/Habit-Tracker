# Week 1 — Learning-to-learn & Memory (Mit) — print this

**Goal of the week:** own a 13-stop memory palace + first "whoa" moment with a 10-item list.
**Kid time:** 15 min/day after snack. **Parent time:** 5 min at breakfast + Sunday 15 min.

## The week

| Day | Mit does (15 min) | Parent does |
|---|---|---|
| Mon | Walk the house with paper. Pick 13 spots in a fixed route (front door → … → his bed). Number them, draw the map. | Nothing. Don't help pick the spots — his route, his palace. |
| Tue | Eyes closed: walk the route 3× forward, 3× backward. Redraw map from memory, check against Monday's. | Ask at dinner: "spot #7 — what is it?" (any random number). |
| Wed | Linking drill #1: get the 10-item list. Link item 1→2→3… with ridiculous MOVING images (banana crushing the couch, not sitting on it). | Write a silly 10-item list on an index card. Hand it over, walk away. |
| Thu | **Breakfast: recall Wednesday's list out loud, in order.** Evening: linking drill #2, new list. | Breakfast test — count correct, no coaching. Write list #2, weirder. |
| Fri | Breakfast: recall list #2. Evening: recall it BACKWARDS. Then walk the palace once, forward + backward. | Breakfast test. At dinner, have him do the backwards recall for the whole family. |
| Sun | Sunday session (below). | Run it. |

## Sunday session (15 min)

1. Numbers on the fridge chart: palace stops mastered ( /13), list #1 score ( /10), list #2 score ( /10)
2. Celebrate the STRATEGY: "the moving images — that's exactly what memory champions do"
3. Next week's micro-goal: palace #2 (grandma's house or school route), Memory League free account
4. If the backwards recall happened → add it in /journey/admin ("memorized 10 items, recalled backwards, week 1 of memory training") → shows on his /trophies. Let him watch you type it.

## Rules

- Reread to study = banned. Guess first, then check. Every time.
- One drill per day. Don't binge; the spacing IS the technique.
- If a recall fails: "which image went dark? Make it louder." Never just re-tell him the item.

**Next week:** palace #2 + first list stored IN the palace. Week 3: Memory League ladder starts.
