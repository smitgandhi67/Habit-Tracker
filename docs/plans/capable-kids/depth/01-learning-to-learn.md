# Depth Pack 01 — Learning-to-learn & Memory

**Kid:** Mit (grade 5, 10–11) · **Dose:** 10–15 min/day, 5 days/week · **Cycle:** 12 weeks
**Promise:** by week 12 Mit memorizes a 20-item list in minutes, studies for tests with a system
instead of rereading, and can predict his own quiz scores within ~10%.

---

## 1. Dimension map

"Learning-to-learn" is five separate trainable sub-skills. Train them in this order; each feeds the next.

| # | Sub-skill | What it concretely means | Weekly number |
|---|---|---|---|
| a | Memory technique | Turn anything into images placed in a mental route | Memory League Images level |
| b | Spaced retrieval | Review school facts at expanding intervals, closed-book | Deck retention % |
| c | Calibration | Predict your score before checking; shrink the error | Prediction error (points) |
| d | Plan–monitor–evaluate | Wrap every homework in 1 planning + 1 evaluation question | Days used (0–5) |
| e | Strategy selection | Name WHICH tool fits the material before starting | Sunday quiz: tool named correctly? |

## 2. Technique stack

| Technique | From | Tag | Why it earns its place |
|---|---|---|---|
| Linking / story method | Memory sport, first lesson everywhere | KNOWN | Gateway drug: 10-item list day one, instant "whoa" moment |
| Memory palace (13-locus home route) | Memory sport core; [beginner protocol](https://thecraftofmemory.substack.com/p/a-beginners-guide-to-memory-training) | VERIFIED for memorization itself (technique of every world champion; brain-imaging studies of athletes show method, not gift). No far-transfer claim made. |
| Memory League Images ladder | [memoryleague.com](https://memoryleague.com/) — L1: 6 images/60s → L10: 30 images/60s; Kid division ≤13 | KNOWN | External, objective, game-shaped ladder; free account is enough to start |
| Retrieval practice + expanding intervals (1/3/7) | [Dunlosky 2013] — already in citations registry | VERIFIED | The two highest-utility techniques in the learning literature |
| Anki or paper Leitner box | [Anki manual](https://docs.ankiweb.net/background.html) | VERIFIED (mechanism = retrieval + spacing) | Retention % is a number a kid can watch go up |
| Calibration (predict → check → chart) | [EEF metacognition guidance](https://educationendowmentfoundation.org.uk/education-evidence/guidance-reports/metacognition) (+8 months) | VERIFIED | "Knowing what you know" is the highest-leverage metacognitive sub-skill |
| School application bridge | [Mullen Memory](https://mullenmemory.com/) 4-week method (world champion, med student) | KNOWN | Rule: **master technique on fun content first, then aim it at school** — "you don't want to figure out the technique while learning the material" |

## 3. Ladder L0→L5

| Level | Milestone (objective, checkable) |
|---|---|
| L0 | Walks his 13-locus home palace forwards AND backwards, eyes closed, no hesitation |
| L1 | 10-item list (groceries) via linking — 100% recall next morning, order intact |
| L2 | Memory League Images **level 3**; first school list stored in palace (states, vocab, steps) |
| L3 | 20-card school-content deck at **≥85% retention** across 2 weeks; ML Images level 5 |
| L4 | Calibration error **<10%** on 4 straight predictions; palace used for a real test, result on the fridge |
| L5 | ML Images **level 7** or a played Kid-division match; teaches the palace technique to someone else start-to-finish |

L5's "teach it" is deliberate — teaching is the proof of ownership and feeds Pack 02.

## 4. Daily dose protocol (by phase)

**Weeks 1–2 — Foundations (no app yet)**
- Day 1–2: build the palace. Walk the house, pick 13 loci in a fixed route (front door → kitchen →
  … → his bed). Draw the map on paper. Walk it mentally 3× forward, 3× backward.
- Day 3–5: linking drill. Parent writes a 10-item silly list; Mit links each item to the next with
  exaggerated, moving, ridiculous images. Test next morning at breakfast (that's the spacing).
- Rule from the pros: **two images per locus, interacting with the locus** (the banana is crushing
  the couch, not sitting on it). Movement + exaggeration = stickiness.

**Weeks 3–6 — The ladder (Memory League, free account)**
- Beginner schedule from the memory-sport community, fits 15 min exactly:
  - Mon: 1 Images attempt (palace #1)
  - Tue: build palace #2 (13 more loci — grandma's house, school route), review both
  - Wed: 1 attempt (palace #2)
  - Thu: 1 attempt (palace #1)
  - Fri: review both palaces + 1 attempt, log the level
- Parallel from week 4: first **school deck** — 15–20 cards from actual current unit (science terms,
  Gujarati words, state capitals). Review day 1/3/7. Paper Leitner box works if screens are a concern.

**Weeks 7–12 — Aim it at school + calibrate**
- 2 days/week: Memory League ladder (keep climbing, it's the fun part)
- 2 days/week: school material into palace or deck — whatever test is actually coming
- Every deck review and every real quiz: **predict the score first, out loud or on the chart, then check.**
  Chart lives on the fridge: predicted vs actual, one dot per event.
- Homework wrapper every day (60 seconds, not a session): before — "what's your plan?" after —
  "what worked, what would you change?"

## 5. Parent playbook

**Sunday session (15 min, same slot weekly):**
1. Record the four numbers (ML level, retention %, calibration error, wrapper days) — 2 min
2. One specific celebration — name the strategy, not the talent: "linking the images with motion —
   that's exactly what the pros do," never "you're so smart"
3. Set ONE micro-goal for the week ("level 4 by Friday")
4. If something real happened (first palace, first ML level-up, aced quiz he predicted) → add it in
   /journey/admin → it lands on his /trophies. Let him watch you type it.

**Mid-week parent drills (budget allows 2+ hr/week):**
- Breakfast recall test of last night's list (30 seconds, huge signal value)
- "Quiz me" reversal: Mit builds 5 questions from his unit and quizzes YOU; you model thinking
  aloud, including being wrong and calibrating ("I said 80% sure and I was wrong — interesting")
- Be the list-writer for linking drills; make the lists progressively weirder

**In-the-moment lines:**
- Frustration at a failed recall: name it first — "that's annoying, you had it yesterday" — THEN
  "which locus went dark? Let's make that image louder." Never skip to the fix.
- Wants to reread instead of retrieve: "close the book, guess first, then check." Every time.
- Don't rescue mid-recall. Silence is the technique working.

## 7. Performance outlet

- **Week 6:** family demo night — Mit memorizes a 15-item list the audience shouts out, recalls it
  backwards. (Backwards recall is the crowd-killer and is trivial with a palace.)
- **Week 12:** a real Memory League match (online, anytime) — Kid division is ≤13, so he's eligible
  for actual competition whenever ready. Win or lose, the match itself goes on the brag sheet.
- **Ongoing:** every aced test where he *predicted* the ace goes in /journey/admin. The trophy is
  the calibration, not the grade.
