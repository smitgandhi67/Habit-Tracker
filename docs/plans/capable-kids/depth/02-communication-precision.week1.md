# Week 1 — Communication Precision (Mit) — print this

**Goal of the week:** two baselines locked away + describe-and-draw becomes a family game.
**Time:** 10 min at dinner, 4 nights. Runs alongside the memory pack (that one is solo after-snack; this one is family-at-dinner).

## The week

| Day | What happens (10 min at dinner) |
|---|---|
| Mon | **Baseline 1:** Mit describes a hidden 4-shape figure (parent draws only what the words say). Score = shapes right /4. Write the % down. **Baseline 2:** phone out — Mit retells any movie he likes for 1 minute. Save the video. DO NOT delete it, do not critique it. It's the week-12 before/after. |
| Tue | Describe-and-draw: Mit describes, parent draws. 3-shape figure. Rules: no hands, no peeking until done. Score it together. |
| Wed | Swap: parent describes, Mit draws. (Hearing a good description teaches more than giving a bad one.) Then one more with Mit describing — 4 shapes if Tuesday was ≥80%. |
| Thu | Sibling or other parent as drawer. Mit describes. Add a shape if he scored ≥80% yesterday. |
| Fri | Family round: everyone describes one figure, everyone gets scored. Mit keeps the scoreboard. |
| Sun | Sunday session (below). |

## Making figures (10 seconds each)

Draw 3–5 overlapping shapes on an index card: circle, triangle, square, arrow, line. Vary size and
position ("big circle, small triangle sitting on top of it, arrow through both pointing left").
Whoever describes hasn't seen anyone else's card.

## Sunday session (5 min — shares the slot with the memory pack's session)

1. Fridge chart: best describe-and-draw % this week
2. Celebrate precision, specifically: "'triangle POINTING DOWN, touching the circle's top' — that's why dad drew it right"
3. Next week's micro-goal: 5-shape figures at 80%
4. Baseline videos confirmed saved (both packs' brag entries go into /journey/admin together)

## Rules

- The drawing is the score. No arguing with the paper.
- One re-try max on anything. Reps beat perfection.
- Never correct grammar mid-game. This is about the picture crossing the table in words.

**Next week:** 5-shape figures + learn the 5-finger story frame (thumb who/where · index problem ·
middle tried · ring ended · pinky zinger). Week 3: recorded 1-minute retells start.
