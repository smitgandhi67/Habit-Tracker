# Depth Packs — framework

> Why this exists: the Skills module has 10 broad domains and a one-line activity library.
> Breadth without depth doesn't produce growth. A Depth Pack takes ONE dimension and gives it
> a real curriculum: technique, ladder, daily dose, parent script, and a public performance target.
> Built for a working parent — the kid runs the daily practice; the parent reviews and celebrates.

## The thesis: extraordinary via small doses

Small daily practice reaches extraordinary levels only in **technique-leveraged** dimensions:

```
performance = learnable technique × accumulated reps × fast feedback
```

…and almost nobody trains the technique. That is the arbitrage. A 10-year-old with 12 weeks of
memory-palace practice out-memorizes adults not by talent but because adults never learned the
technique. Same pattern: touch typing, mental math, chess tactics, structured speaking.

**The dose science says 10–20 min/day is correct, not a compromise.** Ericsson's deliberate-practice
research: children's programs start at 10–20 minute sessions; benefits fall off past ~1 hour/day even
for adults. The conditions that matter more than duration:

1. **Specific micro-goal** for the session ("reach level 3", not "practice memory")
2. **Full attention** — no background screen, short enough that focus holds
3. **Immediate feedback** — app score, recording, parent check, answer key
4. **Just beyond reach** — the level system does this automatically

Source: [Deliberate Practice — original definition & limits (Macnamara et al.)](https://pmc.ncbi.nlm.nih.gov/articles/PMC6824411/)

## Evidence tags

Same convention as `server/capabilities/citations.js`:

| Tag | Meaning |
|---|---|
| VERIFIED | RCT / meta-analysis / major guidance body (EEF, Dunlosky) |
| MIXED | Works for the trained skill; far-transfer claims weak or contested |
| KNOWN | Community-standard practice, no formal trial; low risk, clear mechanism |

Rule inherited from the module: no unsourced "brain training" claims. If a technique only improves
the trained game, say so and value it for what it actually delivers.

## Pack template (7 sections)

1. **Dimension map** — 4–6 concretely trainable sub-skills, not prose
2. **Technique stack** — stolen from the expert/competition community, each tagged + sourced
3. **Ladder L0→L5** — objective milestones, externally benchmarked where possible
   (Memory League level, retention %, rubric score on a recording). Progress must be visible
   without parental judgment.
4. **Daily dose protocol** — exact 10–15 min drill per ladder stage; feedback source named;
   kid-self-running by default
5. **Parent playbook** — Sunday 15-min session script, mid-week parent-led drills, and
   in-the-moment lines (name the feeling → validate → then solve; praise strategy, not talent;
   don't rescue)
6. **Measurement** — one number per sub-skill per week, written down every Sunday
7. **Performance outlet** — a real audience or competition on the calendar. Skills grown in
   private stay private; the outlet is what makes it real to the kid.

## Operating rhythm (all packs share it)

- **Daily (kid, 10–15 min):** the dose. Same time slot every day — attach to an existing anchor
  (after snack, before dinner).
- **Weekly (parent + kid, Sunday, 15 min):** record the numbers, celebrate one thing specifically
  ("you held 26 loci — that's a real palace now"), set next week's micro-goal, capture any real
  achievement into /journey/admin → it appears on the kid's /trophies.
- **Every 12 weeks:** performance outlet event, then rotate or level-up the pack.
